# EduOne | UI Pages Structure

This document describes all main pages and user flows for the platform.

---

## 1) Landing Page (Before Login)
- Platform name and slogan
- Short explanation of how EduOne works
- CTA buttons:
  - Create Account
  - Login
  - Browse Teachers
- Section for teachers to join EduOne

---

## 2) Authentication
### Login
- Email
- Password
- Forgot password

### Register
- Full name
- Email
- Password
- Role selection (Student / Teacher)

---

## 3) Student Pages
### Student Dashboard
- Upcoming sessions
- Quick access to booking
- Notifications

### Browse Teachers
- Search
- Filters (subject, price, availability, rating)
- Teacher cards

### Teacher Profile
- Bio
- Subjects
- Session types available
- Prices
- Availability
- Book session button

### Booking Page
- Select date & time
- Select session type
- Notes
- Confirm booking

### My Bookings
- Upcoming
- Past
- Session status

### Session Page
- Session details
- Join button (online)
- Location info (in-person)
- Chat
- Files

---

## 4) Teacher Pages
### Teacher Dashboard
- Booking requests
- Upcoming sessions
- Earnings (later)

### Availability Management
- Set available days & times
- Enable/disable session types

### Booking Requests
- Accept / Reject in-person bookings

### Reviews
- View student reviews
- Average rating

---

## 5) Admin Pages
- Dashboard overview
- Manage users
- Verify teachers
- Manage subjects
- Monitor bookings
- Moderate reviews
